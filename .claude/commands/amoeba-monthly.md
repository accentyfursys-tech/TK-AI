# 채산 월 결산 자동화

아메바 어카운트에서 다운받은 분개장 로우데이터를 자동 분류하고 피봇 보고서를 생성합니다.

## 트리거 키워드
"채산 결산", "분개장 처리", "아메바 월결산", "피봇 생성", "채산피봇"

## 실행 전 확인사항
- `02. 분개장정보상세.xlsx` 파일이 아래 경로에 있는지 확인
- 경로: `C:/iloom-workspace/10-projects/12-amoeba-tk/amoeba monthly/`
- 처리완료 파일(`02. 분개장정보상세_처리완료.xlsx`)이 열려있으면 닫기

## 실행 순서

### Step 1. RAW 데이터 처리
```bash
cd "C:/iloom-workspace/10-projects/12-amoeba-tk/amoeba monthly"
python update_raw.py
```
→ 원본 분개장에 분류 컬럼 추가 후 `_처리완료.xlsx` RAW 시트 생성

### Step 2. 채산피봇 (스타일 보고서)
```bash
python make_styled_pivot.py
```
→ 고정비/변동비/영업외손익 색상 구분된 정적 피봇 시트 생성

### Step 3. 엑셀피봇(표1) + 분석피봇
```bash
python make_pivots.py
```
→ 채산계정3 상세 피봇 + 감가상각/물류비/기타관리비 분석 피봇 생성

### Step 4. 채산피봇(라이브)
```bash
python make_live_pivot.py
```
→ RAW와 연결된 Excel 실제 피봇테이블 생성

## 주요 분류 로직

### 직접/간접 (채산코스트센터 기준)
- **직접**: 유통경쟁력강화팀, 온라인CX개선팀, 리테일, 직영+리테일, 영업효율, 일룸유통
- **간접**: 그 외 전체

### 채산계정3 세부 분류
- **물류비(물류/운송)**: 비고(적요)에 '안성' 포함 → _시디즈 / 나머지 → _바로스
- **판매수수료**: PSA / PG사(SPC·페이) / 외부몰(쿠팡·네이버·29CM 등) / 입점몰(롯데·스타필드·신세계·송도) / 투자B / B/S / 기타

### 팀세부 (안분된채산조직 기준)
- 온라인CX개선팀 → 온라인
- 일룸유통 → 리테일
- 일룸직영 → 직영

### 세부
- 안분된채산조직의 괄호`()` 안 값 추출

### 기타일반관리비
- **간접항목**: 교육훈련비, 도서인쇄비, 보험료, 비품비, 소모품비, 수도광열비, 여비교통비, 접대비, 차량유지비, 통신비
- **직접항목**: 도서인쇄비, 경상연구개발비, 보험료, 세금과공과, 소모품비, 차량유지비, 하자보수비

## 피봇 구조
- **열 순서**: 리테일 → 직영 → 온라인 (각 간접/직접)
- **행 순서**: 고정비 → 변동비 → 영업외손익
- **숫자 형식**: 원단위, 천원 쉼표(#,##0)
