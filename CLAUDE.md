## 작업 규칙

### Claude Code 사용 시
1. **Escape 키**: 작업 중단
2. **/clear**: 컨텍스트 초기화
3. **/compact**: 긴 대화 압축
4. **@ 참조**: 파일 직접 참조 (`@파일명.md`)
5. **Tab**: 자동완성

### 파일 작업
- 기존 파일 편집 우선 (새 파일 생성 최소화)
- 날짜 형식: `YYYY-MM-DD_제목.md`
- 폴더 이동 시 경로 확인

### Git 규칙
- 작은 단위로 자주 커밋
- 커밋 메시지는 한글로 간결하게
- push 전 확인

## 자주 쓰는 프롬프트

```

```
