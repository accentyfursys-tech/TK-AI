## RAW 추가 컬럼 및 분류 로직

### 1. 직접간접 (K열, 공통직접구분 다음)
- 기준: `채산코스트센터`
- **직접** 키워드: 유통경쟁력강화팀, 온라인CX개선팀, 리테일, 직영+리테일, 영업효율, 일룸유통
- 나머지 → **간접**

### 2. 팀세부 (P열, 안분된채산조직 앞)
- 기준: `안분된채산조직`
- 온라인CX개선팀 포함 → 온라인
- 일룸유통 포함 → 리테일
- 일룸직영 포함 → 직영

### 3. 세부 (R열, 안분된채산조직 바로 뒤)
- 기준: `안분된채산조직`
- 괄호`()` 안의 값 추출 (예: `일룸유통(투자B)` → `투자B`)

### 4. 채산계정3 (기존 G열 재가공)
원본값을 `채산계정3_원본`으로 보존 후 아래 로직 적용:

| 원본값 | 조건 | 결과 |
|--------|------|------|
| (판)물류비(물류) | 비고(적요)에 '안성' 포함 | (판)물류비(물류)_시디즈 |
| (판)물류비(물류) | 그 외 | (판)물류비(물류)_바로스 |
| (판)물류비(운송) | 비고(적요)에 '안성' 포함 | (판)물류비(운송)_시디즈 |
| (판)물류비(운송) | 그 외 | (판)물류비(운송)_바로스 |
| (판)판매수수료 (채산계정2 기준) | 비고에 PSA | (판)판매수수료_PSA |
| (판)판매수수료 | 비고에 SPC 또는 페이 | (판)판매수수료_PG사 |

| (판)판매수수료 | 비고에 롯데·스타필드·신세계·송도 | (판)판매수수료_입점몰 |
| (판)판매수수료 | 세부에 '투자' | (판)판매수수료_투자B |
| (판)판매수수료 | 세부에 '일반' | (판)판매수수료_B/S |

## 피봇 구조

### 채산피봇 / 채산피봇(라이브) 공통
- **열**: 리테일(간접/직접) → 직영(간접/직접) → 온라인(간접/직접) → 총합계
- **행**: 고정비 → 변동비 → 영업외손익 (채산계정3 상세)
- **색상**: 고정비(초록) / 변동비(파랑) / 영업외손익(빨강)
- **숫자**: 원단위, 천원 쉼표 (#,##0)

### 분析피봇 (3개 피봇)
1. **감가상각피봇** - 유형자산감가상각비 / 무형자산상각비 × 팀세부
2. **물류비피봇** - 물류비 항목 상세 × 팀세부
3. **기타관리비피봇** - 기타일반관리비_간접/직접 × 팀세부

## 2026-04-03 추가 작업: 상제품매출상세 매출구분 분류 로직 수정

### 배경
`01. 상제품매출상세 (2월찐).xlsx` RAW 시트의 매출구분 분류 로직(`transform_raw5.py`) 점검 후 3가지 수정 필요 확인.

### 수정 내용

| # | 조건 | 변경 전 | 변경 후 |
|---|------|---------|---------|
| 1 | 상차지창고 == 일룸전시품창고 | 이름prefix 있으면 일반으로 빠짐 | 무조건 전시품 (이름prefix 체크 전에 처리) |
| 2 | 건명에 '전시품 시공병행' | 공백 있는 경우만 인식 | 괄호 여부 무관하게 모두 → 일반 (`re.search` 정규식 적용) |
| 3 | 건명에 '영업장구' | 미처리 (일반으로 분류됨) | 전시품으로 분류 (신규 추가) |

### 우선순위 (수정 후)
```
extra → 판촉물 → 전시품창고(무조건) → 이름prefix→일반 → 영업장구→전시품 → 시공병행→일반 → 전시품/레이아웃→전시품 → 일반
```

- 수정 스크립트: `transform_raw5.py`
