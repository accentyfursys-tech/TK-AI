### 2. 열 추가 및 구조 변경 (35열 → 41열)

#### E열 - 매출구분 삽입
- 위치: `매출일자` 다음, `건명` 앞
- 값: 건명 기반 분류 로직으로 채움 (아래 분류 기준 참고)

#### 실적사업소 다음 - 채널구분 삽입
- 위치: `실적사업소` 바로 다음
- 값: `안분된채산조직` 기준 분류

| 안분된채산조직 | 채널구분 |
|---|---|
| 일룸유통 | 오프라인 |
| 일룸직영 | 오프라인 |
| 온라인CX개선팀 | 온라인 |

#### 안분된채산조직 - 괄호 제거
- 기존: `일룸유통(일반)`, `일룸직영(송파)`, `온라인CX개선팀(공식몰)(일룸)`
- 변경: `일룸유통`, `일룸직영`, `온라인CX개선팀`

#### 안분된채산조직 다음 - 세부 열 삽입
- 위치: `안분된채산조직` 바로 다음
- 값: 괄호 안 내용 추출 (`(일룸)` 은 무시)

| 예시 | 세부 |
|---|---|
| 일룸유통(일반) | 일반 |
| 일룸유통(투자) | 투자 |
| 일룸직영(송파) | 송파 |
| 온라인CX개선팀(공식몰)(일룸) | 공식몰 |
| 온라인CX개선팀(외부몰)(일룸) | 외부몰 |

#### 안분된채산조직_계층 - 3열 세분화
- 위치: `안분된채산조직_계층` 바로 다음
- `//` 구분자로 분리

| 열명 | 예시 값 |
|---|---|
| 채산조직_사업부 | 일룸사업부 |
| 채산조직_팀 | 리테일사업팀(공통)(일룸) |
| 채산조직_채널 | 일룸유통(일반) |

### 3. 매출구분 분류 기준

| 구분 | 조건 |
|---|---|
| **extra** | 건명이 빈값이거나, `매출조정` 포함 (단, `전시품`도 포함되면 제외) |
| **판촉물** | 건명에 `판촉물` 포함 |
| **전시품** | 상차지창고 = `일룸전시품창고` (무조건) / 건명에 `전시품` 또는 `(레이아웃)` 포함 |
| **일반** | 위 모두 해당 없는 경우 |

**전시품 제외 조건:**
- 건명 앞에 사람 이름(한글 2~4글자)으로 시작하는 경우 → **일반**
  - 예: `염현희(전시품)`, `최송화(전시품)(현)`, `이경호(전시품)` 등
- 건명에 `전시품 시공병행` 포함 → **일반**

**분류 결과:**
| 매출구분 | 건수 |
|---|---|
| 일반 | 249,574 |
| 전시품 | 921 |
| extra | 34 |
| 판촉물 | 7 |
| **합계** | **250,536** |
