# 주간 채산 데이터 추출 사용법

## 매주 할 일 (3단계)

### 1단계. 아메바어카운트에서 엑셀 다운로드
- 아메바어카운트 실행
- 전주 데이터 조회 후 엑셀 다운로드
- 저장 위치: `C:\iloom-workspace\10-projects\12-amoeba-tk\amoeba weekly`

### 2단계. 채산추출_실행.bat 더블클릭
- **파일 탐색기(폴더 아이콘)** 열기
- `C:\iloom-workspace\10-projects\12-amoeba-tk\amoeba weekly` 폴더 이동
- `채산추출_실행.bat` 더블클릭
- 결과 확인 (전주 매출 / 매출원가 / 비율 자동 계산)
- **클립보드에 자동 복사됨**

### 3단계. 컨플루언스에 붙여넣기
- 컨플루언스 보고 양식 열기
- 해당 주차 셀에 Ctrl+V

---

## 결과 예시
```
매출        6,131 (백만원)   100.0%
매출원가    3,010 (백만원)    49.1%
```

---

## 주의사항
- bat 파일은 반드시 **파일 탐색기에서 더블클릭** (VS Code에서 클릭 X)
- 엑셀 파일은 반드시 `amoeba weekly` 폴더에 저장
- 엑셀 파일이 여러 개면 **가장 최근 파일** 자동 선택
